# Consulting Crew Website

Professional consulting company website built with HTML, CSS, and JavaScript.

## Features
- Responsive design (mobile-first)
- Header/Footer includes system
- Live chat widget
- Back to top button
- Cookie consent banner
- Professional service pages
- Blog/Insights section
- Legal pages (Privacy, Cookie, Terms)

## Deployment
1. Push to GitHub repository named `username.github.io`
2. GitHub Pages will automatically deploy
3. Visit `https://username.github.io`

## Local Development
Open any HTML file directly in browser or use a local server:
```bash
# Using Python
python -m http.server 8000
